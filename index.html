Projects (Selected)

Physically Based Rendering - 2015

In this project, I implemented a multi-threaded physically based renderer in C++. I started with a provided basic UI framework that showed a window for rendering result output. My main contribution was the implementation of various physically based rendering techniques. The main features of the render includes: Photon Mapping, Monte Carlo Path Tracing, BVH acceleration tree with surface area heuristic, BSDF materials (e.g. glass), and Depth of Field.
